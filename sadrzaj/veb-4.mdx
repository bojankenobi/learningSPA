## Nedelja 5: Funkcije (Functions)

Zamislimo da imamo formulu za računanje površine pravougaonika koju moramo da iskoristimo 50 puta na sajtu. Da ne bismo pisali istu formulu 50 puta, mi je "zapakujemo" u funkciju.

Funkcija je kao mini-mašina:
1. **Ulaz (Parametri):** Šta ubacujemo u mašinu (npr. širina i visina).
2. **Obrada:** Šta mašina radi sa tim podacima.
3. **Izlaz (Return):** Šta nam mašina vraća kao rezultat.

Ovo prati najvažnije pravilo programiranja: **DRY (Don't Repeat Yourself - Ne ponavljaj se)**.

```javascript
// 1. Deklaracija funkcije
function izracunajPovrsinu(sirina, visina) {
    let povrsina = sirina * visina;
    return povrsina; // Vraća rezultat onome ko je pozvao funkciju
}

// 2. Pozivanje (izvršavanje) funkcije
let soba1 = izracunajPovrsinu(4, 5); // Ubacujemo 4 i 5, dobijamo 20
let soba2 = izracunajPovrsinu(10, 3); // Ubacujemo 10 i 3, dobijamo 30

console.log(`Površina prve sobe je ${soba1} kvadrata.`);
console.log(`Površina druge sobe je ${soba2} kvadrata.`);
```

## Nedelja 6: Nizovi (Arrays)

Do sada je jedna varijabla čuvala jednu vrednost (`let ime = "Ana"`). Ali šta ako imamo spisak od 100 imena? Tu koristimo **Nizove**.

Niz (Array) je lista vrednosti smeštena u jednu promenljivu. Piše se unutar srednjih zagrada `[ ]`.
**Važno:** U programiranju, brojanje (indeksiranje) uvek počinje od nule (0), a ne od jedan!

```javascript
// Kreiranje niza
let spisakZaKupovinu = ["Mleko", "Hleb", "Jaja"];

// Čitanje iz niza (Indeksi kreću od 0)
console.log(spisakZaKupovinu[0]); // Ispisuje "Mleko"
console.log(spisakZaKupovinu[1]); // Ispisuje "Hleb"

// Dodavanje nove stavke na KRAJ niza (push)
spisakZaKupovinu.push("Kafa");
console.log(spisakZaKupovinu); // ["Mleko", "Hleb", "Jaja", "Kafa"]

// Brisanje poslednje stavke (pop)
spisakZaKupovinu.pop(); 

// Provera koliko elemenata ima u nizu (length)
console.log(`Na spisku ima ${spisakZaKupovinu.length} stvari.`);
```

## Nedelja 7: Objekti (Objects)

Dok su nizovi dobri za proste liste, **Objekti** su savršeni za modelovanje stvarnog sveta. 
Objekat grupiše osobine koje pripadaju jednom entitetu. Piše se unutar vitičastih zagrada `{ }` i sastoji se od parova **ključ: vrednost** (key-value).

Objekti čak mogu imati i svoje funkcije, koje se tada zovu **metode**.

```javascript
// Kreiranje objekta
let masina = {
    model: "X-2000",
    brzina: 1500,
    status: "Ugašena",
    
    // Metoda (Funkcija unutar objekta)
    pokreni: function() {
        this.status = "Uključena"; // 'this' označava OVAJ objekat
        console.log(`Mašina ${this.model} je sada ${this.status}.`);
    }
};

// Pristupanje podacima preko tačke (.)
console.log(`Model mašine je: ${masina.model}`);
console.log(`Trenutna brzina: ${masina.brzina} RPM`);

// Pozivanje metode
masina.pokreni();
```

## Nedelja 8: Iteracija kroz složene podatke



Prava moć JavaScript-a leži u kombinovanju svega što smo do sada naučili: petlji, if/else logike, nizova i objekata.
U realnim aplikacijama (poput Facebook-a ili baze zaposlenih), podaci nam najčešće stižu kao **Niz objekata**. Pomoću `for` petlje, možemo proći kroz svaki objekat na listi i uraditi nešto sa njim.

```javascript
// NIZ koji u sebi sadrži OBJEKTE
let zaposleni = [
    { ime: "Marko", pozicija: "Operater", naSmeni: true },
    { ime: "Jelena", pozicija: "Inženjer", naSmeni: false },
    { ime: "Ivan", pozicija: "Tehničar", naSmeni: true }
];

console.log("--- Radnici trenutno na smeni ---");

// Idemo od 0 do kraja niza
for (let i = 0; i < zaposleni.length; i++) {
    // Ako je radnik trenutno na smeni (true)
    if (zaposleni[i].naSmeni === true) {
        console.log(`${zaposleni[i].ime} (${zaposleni[i].pozicija})`);
    }
}
```
