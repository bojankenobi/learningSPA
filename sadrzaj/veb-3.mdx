# 3. JS Modul 1: Temelji i Logika

Ako je HTML kostur, a CSS koža i odeća naše veb stranice, onda je **JavaScript (JS)** njen mozak i mišićni sistem. Bez JS-a, stranica je samo lepa slika. Sa JS-om, ona postaje interaktivna aplikacija koja može da računa, donosi odluke i komunicira sa serverima.

U ovom prvom modulu, zaboravićemo nakratko na HTML i CSS. Fokusiraćemo se isključivo na to **kako JavaScript razmišlja** i kako računar obrađuje podatke. Sve primere ćemo pokretati u "Konzoli" vašeg brauzera (pritisnite `F12` na tastaturi i otvorite tab *Console*).

## Nedelja 1: Uvod i Varijable (Promenljive)

Računar mora negde da zapamti podatke dok radi sa njima. Za to koristimo **varijable**. Možete ih zamisliti kao kutije sa imenima u koje ubacujemo vrednosti.



U modernom JavaScript-u koristimo dve ključne reči za pravljenje ovih "kutija":
* `let`: Koristimo kada znamo da će se vrednost u kutiji **menjati** (npr. rezultat u igrici, broj godina).
* `const` (konstanta): Koristimo kada vrednost **ne sme da se menja** (npr. datum rođenja, PDV stopa).
* *(Napomena: Nekada se koristila reč `var`, ali je danas izbegavamo jer može izazvati bagove u većim programima).*

### Primer: Osnovna matematika u JS-u

```javascript
// Deklaracija varijabli
let imeKorisnika = "Marko";
const godinaRodjenja = 2005;
let trenutnaGodina = 2024;

// Računanje
let brojGodina = trenutnaGodina - godinaRodjenja;

// Ispis u konzolu (Console)
console.log("Zdravo, " + imeKorisnika + "!");
console.log("Ti imaš " + brojGodina + " godina.");

// Promena vrednosti (Ovo radi jer smo koristili 'let')
trenutnaGodina = 2025;
brojGodina = trenutnaGodina - godinaRodjenja;
console.log("Sledeće godine imaćeš " + brojGodina + " godina.");

// godinaRodjenja = 2000; // OVO BI IZBACILO GREŠKU! 'const' se ne menja.
```

## Nedelja 2: Tipovi podataka i Operatori

Kutije (varijable) mogu da čuvaju različite vrste podataka:
1. **String** (Tekst): Uvek ide pod navodnike `"tekst"` ili `'tekst'`.
2. **Number** (Broj): Piše se normalno `42` ili `3.14`.
3. **Boolean** (Logička vrednost): Može biti samo `true` (tačno) ili `false` (netačno).

### Operatori poređenja i Logički operatori
Da bi računar doneo odluku, on mora da uporedi vrednosti.
* `===` (Strogo jednako) - Da li su dve vrednosti potpuno iste?
* `!==` (Nije jednako)
* `>`, `<`, `>=`, `<=` (Veće, manje, veće ili jednako...)
* `&&` (I / AND) - Oba uslova moraju biti tačna.
* `||` (ILI / OR) - Barem jedan uslov mora biti tačan.

### Primer: Template Literals (Spajanje teksta i varijabli)
U modernom JS-u, za spajanje teksta koristimo "backticks" (\`) i znak `${}`. Ovo je mnogo čistije od korišćenja znaka `+`.

```javascript
let proizvod = "Laptop";
let cena = 85000;
let naStanju = true;

// Strogo poređenje (vraća true ili false)
console.log(cena > 50000); // true (cena jeste veća)
console.log(cena === 85000); // true

// Spajanje stringova (Template Literal)
let poruka = `Ime proizvoda je ${proizvod}, a cena je ${cena} RSD. Da li je na stanju? ${naStanju}`;

console.log(poruka);
```

## Nedelja 3: Kontrola toka (If/Else)

Do sada se naš kod izvršavao liniju po liniju, od vrha do dna. Ali šta ako želimo da se neki kod izvrši **samo ako** je određeni uslov ispunjen? Tu na scenu stupa `if/else` blok.

Program donosi odluke baš kao i mi: *"Ako (if) pada kiša, poneću kišobran. Inače (else), poneću naočare za sunce."*

### Primer: Provera godina korisnika

```javascript
let korisnikGodine = 16;
let imaVipKartu = false;

// Da li korisnik sme da uđe u noćni klub?
if (korisnikGodine >= 18) {
    console.log("Dobrodošli! Imate pravo pristupa.");
} else if (imaVipKartu === true) {
    console.log("Nemate 18 godina, ali imate VIP kartu. Ulaz je odobren.");
} else {
    // Ako nijedan gornji uslov nije ispunjen, izvršava se ovaj blok
    let faliGodina = 18 - korisnikGodine;
    console.log(`Pristup odbijen. Vratite se za ${faliGodina} godine/a.`);
}
```

## Nedelja 4: Petlje (For i While) - Osnova automatizacije

Šta ako želimo da ispišemo brojeve od 1 do 100? Nećemo valjda pisati `console.log()` sto puta? Računari su savršeni za dosadne zadatke koji se ponavljaju. Za to koristimo **petlje** (loops).

### For Petlja
Najčešće se koristi kada tačno znamo koliko puta želimo da ponovimo radnju.
Sastoji se iz 3 dela:
1. **Brojač** (npr. `let i = 1`)
2. **Uslov** do kada se vrti (npr. `i <= 5`)
3. **Korak** povećanja brojača (npr. `i++` što znači uvećaj za 1)

```javascript
// Ispisivanje brojeva od 1 do 5
for (let i = 1; i <= 5; i++) {
    console.log(`Ovo je prolaz broj ${i}`);
}

// Ispisivanje samo PARNIH brojeva od 1 do 10
console.log("Parni brojevi:");
for (let i = 2; i <= 10; i = i + 2) {
    console.log(i);
}
```

### While Petlja
Koristi se kada ne znamo tačan broj ponavljanja, već se petlja vrti **sve dok** je neki uslov tačan.

```javascript
let baterija = 3; // 3 procenata

while (baterija > 0) {
    console.log(`Uređaj radi. Baterija: ${baterija}%`);
    baterija--; // Smanjujemo bateriju za 1
}

console.log("Baterija je prazna. Uređaj se gasi.");
```

Podesi do kog broja petlja treba da ide menjanjem vrednosti `limit`.
